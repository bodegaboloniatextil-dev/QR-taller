# QR-taller

Landing page movil para compartir las cuentas bancarias de Taller de Costura Marbelly y copiar los numeros de cuenta sin espacios.

## Archivos

- `index.html`: pagina principal con las cuentas de pago.
- `qr.html`: pagina imprimible que genera un QR hacia `index.html` cuando el sitio esta publicado.

## Publicacion en GitHub Pages

1. En GitHub, abre `Settings > Pages`.
2. En `Build and deployment`, selecciona `Deploy from a branch`.
3. Elige la rama `main` y la carpeta `/root`.
4. Guarda los cambios.

Cuando GitHub Pages publique el sitio, la URL principal sera:

```text
https://bodegaboloniatextil-dev.github.io/QR-taller/
```

El QR estara disponible en:

```text
https://bodegaboloniatextil-dev.github.io/QR-taller/qr.html
```

Tambien se incluye el QR en PNG:

```text
qr-taller-marbelly.png
```
